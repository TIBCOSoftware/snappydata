StreamExecutionThread

Related Doc: package streaming

abstract class StreamExecutionThread extends UninterruptibleThread

A special thread to run the stream query. Some codes require to run in the StreamExecutionThread and will use classOf[StreamExecutionThread] to check.

  32. def runUninterruptibly[T](f: ⇒ T): T

    Run f uninterruptibly in this thread.

    Run f uninterruptibly in this thread. The thread won't be interrupted before returning from f.

    If this method finds that interrupt is called before calling f and it's not inside another runUninterruptibly, it will throw InterruptedException.

    Note: this method should be called only in this thread.
